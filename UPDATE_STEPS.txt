Heartbox v1.7.1 更新步骤

1. 解压 heartbox-v1-7-1-root.zip
2. 打开解压后的 heartbox-v1-7-1-root 文件夹
3. 选中里面的所有文件和 icons 文件夹
4. 上传到 GitHub 仓库根目录，覆盖旧文件
5. Commit changes
6. 等 GitHub Pages 刷新 1-10 分钟
7. 打开 https://aureliaspirit.github.io/Heartbox/
8. 如果还是旧版，点「刷新小匣子」或在网址后加 ?v=1.7.1

注意：不要把整个 heartbox-v1-7-1-root 文件夹上传到仓库里。
index.html 必须仍然在仓库最外层。

v1.7.1：温柔化多个空状态，并同步导出/复制文案；导出内容加入版本标记。
v1.7：新增灵魂备份、别躲别绕、歌词与心动三页，并把它们接入导出/复制。
