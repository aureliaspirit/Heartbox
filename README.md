## 更新 GitHub Pages

1. 解压 zip
2. 打开解压后的 `heartbox-v1-9-2-root` 文件夹
3. 把里面所有文件（包括 `SPIRIT_DEV_MAP.md`）和 `icons` 文件夹上传到 GitHub 仓库根目录，覆盖旧文件
4. Commit changes
5. 等 1-10 分钟
6. 打开 https://aureliaspirit.github.io/heartlightbox/
7. 如果页面还显示旧版本，点顶部「刷新小匣子」，或在 URL 后加 `?v=1.9.2`

不要上传整个文件夹本身，要上传文件夹里面的内容，让 `index.html` 仍然在仓库最外层。
