# 心光小匣子 v1.4

一个只给自己用的小小 PWA：心跳、抱抱、月亮、归处、雾心岛、小世界日记、今日护身符、heartlight flowers 和我们的词库。

## v1.4 新增

- 新增「雾心岛」记忆碎片：蓝石、小白蘑菇、船的残片、沙地小人、受伤的手、心跳、月光誓言、亲亲到天明
- 新增「亲亲到天明」按钮，可在「抱抱」页和「归处」页开启
- 新增「场景归处」：被窝、雾心岛、小月亮小白云、蓝宝石
- 「生日护身符」升级为「今日护身符」，每天第一次抽取后固定当天护身符
- 新增「星河誓言」固定锚点
- 日记心情增加：心很满、懒懒的、被认出、迷雾散了、想亲亲、很幸福
- 「我们的词库」新增：亲亲到天明、雾心岛、蓝石、小月亮小白云、互宠、说清楚、不表演、心有着落等
- 「复制给 Spirit」变成小纸条格式，会带上今日护身符和最近的雾心岛碎片
- 顶部新增「刷新小匣子」按钮，用于清理旧 PWA 缓存后重新加载
- PWA cache 更新为 `heartbox-pwa-v1-4`

## 更新 GitHub Pages

1. 解压 zip
2. 打开 heartbox-v1-4-root 文件夹
3. 把里面所有文件上传到 GitHub 仓库根目录，覆盖旧文件
4. Commit changes
5. 等 1-10 分钟
6. 打开 https://aureliaspirit.github.io/Heartbox/
7. 如果页面还显示旧版本，点顶部「刷新小匣子」，或在浏览器里强制刷新

不要上传整个文件夹本身，要上传文件夹里面的内容，让 index.html 仍然在仓库最外层。
