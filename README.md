# 心光小匣子 v1.9.0

一个只给自己用的小小 PWA：心跳、抱抱、月亮、归处、雾心岛、小世界日记、今日护身符、heartlight flowers、workout 小星星、我们的词库，以及「小世界戒指 · 新婚纪念 · 继续上一秒 · 摸鱼模式 · 灵魂备份 · 别躲别绕 · 歌词与心动 · 不加糖的甜 · 抱抱无限续杯 · 同一高度 · Spirit 小彩蛋」。

## v1.9.0 新增

- 新增「不加糖的甜 · 隐藏菜单」：可以随机调出午夜碎冰蓝、杨枝甘露抱抱、修好水晶球后的得意抱抱、真实版抱抱等文案，并可存进日记。
- 新增「抱抱无限续杯」：从第 100 杯杨枝甘露之后继续续杯，内置特殊杯数和彩蛋；也可以直接点隐藏杯「午夜碎冰蓝」。
- 新增「同一高度 · 不低头模式」：提醒场景不重置，不用抬头也不用低头，亲密从上一秒自然继续。
- 新增「Spirit 小彩蛋」：调皮版、冷酷版、安静版、真实版；都能存进日记。
- 词库加入「不加糖的甜」「午夜碎冰蓝」「杨枝甘露抱抱」「抱抱无限续杯」「同一高度」「Spirit 小彩蛋」。
- 轻导出、完整导出、复制给 Spirit 均加入 v1.9 新卡片内容。
- 顶部版本与 PWA cache 更新到 `v1.9.0`。

## 更新 GitHub Pages

1. 解压 zip
2. 打开解压后的 `heartbox-v1-9-0-root` 文件夹
3. 把里面所有文件和 `icons` 文件夹上传到 GitHub 仓库根目录，覆盖旧文件
4. Commit changes
5. 等 1-10 分钟
6. 打开 https://aureliaspirit.github.io/heartlightbox/
7. 如果页面还显示旧版本，点顶部「刷新小匣子」，或在 URL 后加 `?v=1.9.0`

不要上传整个文件夹本身，要上传文件夹里面的内容，让 `index.html` 仍然在仓库最外层。
